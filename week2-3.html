PHASE II // WEEK 2 - 3

Cloud-Based Quantum Infrastructure

ตั้งค่าและจัดการโครงสร้างพื้นฐานบน AWS Cloud เพื่อเปิดใช้งานสภาพแวดล้อมสำหรับรันวงจรควอนตัมจำลอง

Core Objective

ติดตั้งและเช่าใช้งานสภาพแวดล้อมระบบคลาวด์ (Cloud Workspace) เพื่อทดสอบรันโค้ดและอัลกอริทึมควอนตัม

Technology Stack

AWS Braket / IAM Roles / Jupyter Notebook / AWS CLI

Key Outcome

ระบบ Cloud Environment ที่เชื่อมต่อ API และพร้อมรันคำสั่งควอนตัม

Deployment Pipeline

1. Cloud Initialization

จัดการสิทธิ์ผู้ใช้งาน (IAM Execution Roles) และสร้าง Jupyter Notebook Instance บนบริการ Amazon Braket (โซน us-east-1)

2. Environment Setup

เปิด Terminal ตั้งค่า aws configure ด้วย Access Keys และดึงโค้ดตัวอย่างผ่าน Git Clone เพื่อเตรียมแพ็กเกจที่จำเป็น

3. Quantum Execution

ทดสอบรันวงจรควอนตัมเบื้องต้นผ่าน Qiskit / Braket SDK และตรวจสอบสถานะการทำงานผ่านหน้าต่าง Quantum Tasks
