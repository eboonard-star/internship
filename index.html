Core Tracks

Research Focus Areas

TRACK 01 // OVERVIEW

Conducting Polymers

ศึกษาทฤษฎีกลไกการนำไฟฟ้าของพลาสติก (Conducting Polymer) ตั้งแต่กระบวนการ Doping ไปจนถึงการเกิด Polaron, Bipolaron และ Soliton พร้อมจัดทำเอกสารนำเสนอฉบับย่อย

TRACK 02 // CLOUD LAB

AWS & Quantum

ทดลองเช่าใช้งานและติดตั้งโครงสร้างพื้นฐานคอมพิวเตอร์เสมือน (Virtual Machine) บนคลาวด์ AWS เพื่อเปิดสภาพแวดล้อมในการทดสอบรันชุดคำสั่งควอนตัม

TRACK 03 // DEPLOYMENT

Offline AI (Gemma)

ศึกษาการติดตั้งและตั้งค่าระบบ Large Language Model (LLM) ด้วยโมเดลเปิดของ Google Gemma เพื่อให้สามารถเรียกใช้งานและประมวลผลคำสั่งในระบบแบบปิด

TRACK 04 // LOGIC SYSTEM

Logic Control

สืบค้นและรวบรวมข้อมูลเกี่ยวกับหลักการทำงานของ Logic Control ร่วมกับการสร้างสื่อการนำเสนอ (Slide Material) เพื่ออธิบายตรรกะการควบคุมระบบไอที
